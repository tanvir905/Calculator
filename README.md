# C Calculator

A simple menu-driven calculator program written in C.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Error handling for division by zero

## Concepts Used
- Functions
- switch-case
- Loops
- User input/output

## How to Run
1. Compile the program:
   gcc calculator.c -o calculator

2. Run the program:
   ./calculator

## Sample Output

1. Addition
2. Subtraction
3. Multiplication
4. Division
